# BITES Restuarant

## Description

This is a website made in HTML, CSS, SASS and JS for a made-up restuarant business. Its one of my personal projects for learning purposes and showing off in my portfolio.

## Features

- Responsive
- Navigation
- Multiple pages
- MealDB API Call
- Animated counter
